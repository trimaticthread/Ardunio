# One Light Blink

This is the first Arduino test.
One LED blinks forever.

## File
- one-light-blink.ino

## Wiring
- D5 -> 220R resistor -> LED (+)
- LED (-) -> GND

## What happens
- LED ON for 1 second
- LED OFF for 1 second
- Repeat forever

## If it does not work
- Check LED direction (+ and -)
- Check resistor connection
- Check board and COM port
