# Three LED Sequence

This project lights 3 LEDs one by one.

## File
- two-light-sametimeblink.ino

## Wiring
- LED1 -> D5 (with 220R)
- LED2 -> D6 (with 220R)
- LED3 -> D7 (with 220R)
- All LED negatives -> GND

## What happens
- Step 1: LED1 ON
- Step 2: LED2 ON
- Step 3: LED3 ON
- Each step waits 1 second

## If it does not work
- Check pin numbers
- Check for breadboard short circuits
