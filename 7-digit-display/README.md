# 7-Segment Display

This project shows numbers 0 to 9.

## File
- 7-digit.ino

## Segment pins in this code
- A -> D2
- B -> D3
- C -> D4
- D -> D5
- E -> D6
- F -> D7
- G -> D8
- DP -> D9

## What happens
- The display counts from 0 to 9
- Each number stays 1 second

## Important
If your display type is common-anode, invert ON/OFF logic.

## If it does not work
- Check segment wiring order
- Check common pin wiring
